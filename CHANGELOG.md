# Changelog

## v1.7.0
- 新增子服未启动状态识别，支持 velostat `online` 字段判断子服是否在线
- 新增离线子服橙色主题标识（橙点、橙框、橙字），与查询失败（暗红）区分
- 优化查询失败子服的视觉样式，添加暗红边框、灰名、半透明效果
- 优化代理服务器卡片布局，整体字号放大、间距收紧，提升可读性

## v1.6.0
- 新增代理服务器查询功能，支持查询 Velocity 代理及其子服状态
- 新增 velostat 插件集成，通过 HTTP API 获取子服信息（推荐）
- 新增子服直连查询模式，支持手动配置子服地址列表
- 新增代理服务器状态卡片模板，显示母服+子服列表
- 配置项支持条件显示，根据查询类型动态显示相关配置
