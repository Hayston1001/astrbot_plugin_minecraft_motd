# Changelog

## v1.7.3
- 优化 Java 版直连查询的网络 I/O，将同步 socket 替换为 asyncio 异步 TCP，不再阻塞事件循环
- 修复代理服务器并发查询时事件循环被阻塞的问题，asyncio.gather 现在可以真正并发执行

## v1.7.2
- 修复 MOTD 文本过长导致布局错位的问题，主卡片限4行、代理卡片限2行、子服卡片限单行，超出显示省略号
- 修复代理直连查询时查询失败的子服被静默忽略的问题，现在会显示连接错误信息

## v1.7.1
- 修复子服列表高度不统一的问题，所有卡片等高显示
- 修复 velostat 子服 MOTD 无法显示的问题（字段名映射错误）

## v1.7.0
- 新增子服未启动状态识别，支持 velostat `online` 字段判断子服是否在线
- 新增离线子服橙色主题标识（橙点、橙框、橙字），与查询失败（暗红）区分
- 优化查询失败子服的视觉样式，添加暗红边框、灰名、半透明效果
- 优化代理服务器卡片布局，整体字号放大、间距收紧，提升可读性

## v1.6.0
- 新增代理服务器查询功能，支持查询 Velocity 代理及其子服状态
- 新增 velostat 插件集成，通过 HTTP API 获取子服信息（推荐）
- 新增子服直连查询模式，支持手动配置子服地址列表
- 新增代理服务器状态卡片模板，显示母服+子服列表
- 配置项支持条件显示，根据查询类型动态显示相关配置
